Что такое Ворота деплоймента?

Ворота деплоймента A pre-configured checkpoint or validation in an integration or deployment pipeline that must be passed before promoting code or artifacts to the next stage.

Source: ITIL v4, AWS Well-Architected Framework, Kubernetes Documentation, CNCF

How is “Ворота деплоймента” Used in Practice?
