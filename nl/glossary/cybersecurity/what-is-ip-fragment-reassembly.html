What is IP Fragment Reassembly?

IP Fragment Reassembly The process of reconstructing fragmented IP packets into their original form for delivery, inspection, or security analysis, as described in IETF RFC 791, 815, and NIST SP 800-94.

Source: ISO 27001, NIST Cybersecurity Framework, MITRE ATT&CK

How is “IP Fragment Reassembly” Used in Practice?

Enable deep packet inspection to properly handle IP fragment reassembly and detect evasion techniques that exploit fragmentation.
